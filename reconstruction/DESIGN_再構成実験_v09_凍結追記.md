## 2. 実装時修正記録

**修正1：部位割当方式**
設計書§2.2「各危険マスに攻撃部位を固定割当（6部位均等）」は、D0世界の危険セルが3つのため6部位均等と両立しない。当初は「6部位均等の巡回列を衝突ごとに進める」方式を実装したが、予備実験でG1不成立（FULL一致度0.34）。原因は、同じ場所の傷が衝突順序によって別部位になり、語りが場所と結びつかなくなったこと。**設計原義（マス固定割当）に復帰**し、セル→部位の固定写像（シードごとに急所≥1・末端≥1を保証、決定論生成、乱数消費ゼロ、シード内全条件で共通）とした。この修正後にG1成立。

**修正2：探索フラグの記録**
イベントログに `explored`（その行動がε探索か貪欲かの区別）を追加。§3の測定に必要なため。動態には影響しない（UT-R4で完全継承時のイベント列一致を確認済み）。

## 3. 裁定⑧の顛末（原理は採択・実装は変更なしと実測が示した）

**裁定⑧（始祖・7月24日）**：「記憶の寿命は傷の深さと場所で決まる。急所は末端より長く残る」。
さらに原理への立ち返り：「Pain Memoryの目的は代替経路の探索であり、痛みそのものの禁止ではない」→ **記憶は代替経路が確立するまで生きていればよい**、という基準を採択。

**書記官の測定ミス2回（記録として残す）**
1回目：「同一キーで最初の衝突から最後の衝突まで」を代替経路確立時間とみなす → 中央値838〜872ep。これを根拠に「この世界に代替経路は存在しない」と報告した。**誤り。** この世界には常時安全な迂回路（x=3, y=6のgap_top）が実在する。
2回目：衝突の9割で切っても838ep。同じ誤りを繰り返した。
**原因**：ε=0.2のランダム探索による偶発的な迷い込みを、方策の未確立と誤読していた。学習を測っているつもりで探索ノイズを測っていた。

**3回目（是正後）**：貪欲行動（非探索）での衝突のみを集計。
- 全衝突 615件中、貪欲衝突は114件、探索衝突が501件。
- **同一キーでの2回目の貪欲衝突：0件。** すなわち個体は一度貪欲行動で痛みを負った経路を、二度と自ら選ばない。代替経路の確立は実質1エピソードで完了する。
- 現行λ=0.97は強度1.0の記憶を約115ep保持し、要件を100倍以上のマージンで満たす。→ **急所の減衰延長は不要。λ_vital = λ_own = 0.97 で確定。**
- 部位別減衰の機構自体はコードに残す（他の世界では効きうるため）。

**副産物（探索的観察）**：λ_vital = 0.9987（実質一生もの）を試験実装して走らせたところ、死亡率が 3/10 → 0/10 に低下し、G1も不成立となった。永続化した記憶が永続タブーとなり、個体が危険に接近しなくなり、身体損傷も死も発生しなくなった。これは Pain Memory シリーズが繰り返し示してきた「永久禁止は硬直を招く」の再現であり、始祖の指摘「消えない傷とは代替経路を失うこと」と整合する。検定は行わない。
