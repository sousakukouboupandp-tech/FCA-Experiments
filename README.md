# FCA-Experiments

## これは何か

FCA（Fractal Core Architecture／フラクタルコアアーキテクチャー）は、「AIはどうあるべきか」という設計思想を、工学的な形にまとめた理論です。著者は学者でも研究者でもなく、アーティストです。七宝焼きやシルバーアクセサリーの制作を通じて物質と構造の関係を探求してきた経験から生まれた思想を、AIの設計原理として書き起こしています。

このリポジトリは、その理論の各モジュール（Pain Memoryをはじめとする）について、実際に強化学習のシミュレーションで検証した実験記録の置き場所です。

## 目的

査読論文としての厳密さは目指していません。ここにあるのは、アーティストが自分の手を動かして確かめた実験の記録です。うまくいったこと、いかなかったこと、まだ分かっていないことを、隠さずそのまま残します。

Pain Memoryの核心にある考え方は一つです。**危険な対象そのものを永久に禁止するのではなく、危険に至った経路・文脈を記憶し、時間とともに減衰させ、代替の道を探すことを許す**、という設計です。これが本当にそう機能するのか、様々な状況（火傷、毒、これから増えていく別の危険）で、一つずつ試しています。

## 進め方

- 一つのモジュール・シチュエーションごとに、実験コード・結果・気づいたことをフォルダに分けて残します。
- 予測が外れた実験も、外れたまま記録します。
- 「ここまで確認できた」「ここから先はまだ分からない」を、はっきり区別して書きます。
