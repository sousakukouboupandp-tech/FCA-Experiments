# 幼年期スイープ実験（S(t)＝受信タイミングとシンクロ率）設計書 v01
作成：2026年7月14日／思想：著者（思想注入第3弾・DESIGN_共鳴実験_v01.md §18）／設計：Claude Fable 5
状態：**監査前ドラフト。4AI監査（第1ラウンド）の検品後に事前登録として凍結する。**

## 1. 中心の問い

**「同じ語りでも、聞き手がどれだけ生きたかで、染み方は変わるか。」**

著者の洞察（§18）：受け取るタイミングでシンクロ率Sが変わる。Sは実は時間の関数S(t)である。
これはv04の探索的発見（幼年期後の丸呑みだけが害の方向を示した）から生まれた問いである。

## 2. 設計の骨格

- 環境・パラメータ・エージェント：v04を全凍結流用（変更ゼロ）。A=30シード×1000ep（v04のA_cacheと同一手順）。
- 操作変数はただ一つ：**幼年期の長さ τ ∈ {0, 25, 50, 100, 200, 400} ep**。
  Bは τ ep生きた後にAの語りを受信し、合計1000epまで学習する（総エピソード数は全条件1000で固定）。
- 世界：D0（守りの土俵）と Dbig+（呪いの土俵・v04土俵ゲート合格済み地形）。
- 条件：ResoExp(τ)×6水準。Isolatedは各世界で1本だけ回し、比較時に区間[τ+1, 1000]でスライスして同区間ペア比較する。
- 探索的（主要結論に数えない）：FullCopyDelayed(τ)、τ∈{25,50,100,200,400}、Dbig+のみ。v04の害発見の追跡用。
- 30シード。対応あり設計（同一A記憶・同一B初期シード）。

## 3. 測る数字

- **受信時S̄**：受信時に提示された全S値の平均（ゼロ含む）。受理分（I>0.05）の平均も併記。分布（中央値・四分位・最大）も全公開。
- 衝突エピソード数（区間[τ+1, 1000]）、最短路使用率、成功率、S>0シード数、シード別生データJSON全保存。

## 4. 主要仮説（暫定・監査後に凍結）

- **P1（非飽和）**：D0において、受信時S̄はτ=400がτ=25より大きい（対応ありウィルコクソン、zero_method=pratt、両側、方向を明記）。
  意味：聞き手の成熟は25epで頭打ちにならず、経験を積むほど受信の感度が上がり続ける。
- **P2（薄い経験の門）**：D0において、τ=25のResoExpは同区間[26,1000]のIsolatedより衝突ep数が有意に少ない。
  意味：わずか25epの傷でも、門は開き、守りは発生する。
- 主要検定は2本のみ。ホルム補正。効果サイズrank-biserial併記。

## 7. 自己申告の穴（監査者への正直な白状・5件）

1. **【最重大】P1の半分は設計により真である。** 同一シードのBの軌跡はτによらず途中まで同一であり、傷跡帳はτとともに集合として単調拡大する（τ=400の帳簿⊇τ=25の帳簿）。maxは上位集合で必ず≧となるため、**Sの単調非減少はコードから自明**。測定により真たり得るのは「**厳密に増えるか（飽和点はどこか）**」のみ。P1はこの「非飽和」だけを主張する設計にしたが、この切り分けで十分か監査を求む。
2. **区間交絡**：τが変わると受信後の残りエピソード数が変わる（τ=400は残り600ep）。対Isolated比較は同区間スライスのペアで対処したが、**τ水準間の守り効果の直接比較は交絡が残るため主要検定にしない**。この扱いで十分か。
3. **S̄の定義**：提示平均（ゼロ含む）を主、受理平均を副とした。逆にすべきか。
4. **τ=0は設計により真**（H4で確認済みの構造的ゼロ）のため測定主張から除外。
5. **土俵ゲートの流用**：地形・IsolatedはDbig+でv04ゲート合格済みのため再実行せず、ロードQAのみ再確認とする。手抜きに当たるか。
