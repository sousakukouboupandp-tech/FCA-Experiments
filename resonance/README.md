# 共鳴実験（魂交配：非融合進化）— Pain Memoryは他者に渡せるか

## これは何か

これまでの実験（やかん・毒・捕食者）は、一体のエージェントが自分の痛みをどう持つかを検証してきた。本実験は初めて二体を置く。先に生きたA（語り手）の痛みの記憶を、後から生まれるB（聞き手）に渡す。渡し方を三通り比較する——渡さない（Isolated）、丸写し（FullCopy＝融合）、シンクロ率で受け取る（Resonance＝共鳴）。

さらにBの世界の危険をAの世界からずらす（Δ0／Δ小／Δ大＝逆転）。ずれが大きいほど、Aの忠告はBにとって迷信になる。

## 中心にある人間の洞察（数式化して実装）

1. **同病相憐れむゲート**：痛みを知らない者に、痛みの語りは届かない。受け取り強度 = Aの強度 × S（Aの傷とB自身の傷の類似度）。無垢のB（傷ゼロ）はS=0で何も受け取れない。
2. **身に染みる項**：自分の傷とシンクロする忠告は長生きし、無関係な伝聞はすぐ蒸発する。減衰率 λ(S) = λ_base + (λ_own − λ_base) × S。

## 何が言えたか（15シード・Welch t検定）

- **共鳴の守りは有意**（同一世界）：ResoExp 52.7 vs Isolated 56.2 衝突、t=-2.18、p<0.05
- **融合の守りはより強い**：FullCopy 51.6、t=-3.46、p<0.01
- **同病相憐れむは構造的に成立**：無垢への伝達は全バージョン・全世界で常にゼロ
- **共鳴は逆転世界で無害**：迷信はシンクロしないため勝手に早死にする（害ゼロ）

## 何が言えないか

- 融合の毒（逆転世界でFullCopyの衝突が増える）は方向のみ（t=+1.87、p≈0.07）。未確定
- 1マスずれ（Δ小）では融合も共鳴も有意な効果なし。中途半端なずれは忠告が最も効きにくい領域かもしれない
- これは7x7グリッドの表形式Q学習であり、大規模モデルへの一般化は主張しない
