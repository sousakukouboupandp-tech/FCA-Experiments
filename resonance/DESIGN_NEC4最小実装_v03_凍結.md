## 1. 中心の問い

**「NEC 4.0のLERI式は骨組みか、飾りか。」**

FCA理論への批判ファミリー「複雑なルールの寄せ集め・同語反復」への対抗プログラム第1弾。理論が「これがないと壊れる」と主張する部品を実際に外し、本当に壊れるかを測る。壊れなければ、その部品は飾りだったと公開する。

## 2. 最小実装の完全仕様（16日実装・第三者が同一実装を再現できる水準まで固定）

### 2.1 構造体と出力
- N=32要素。要素iは結合強度w_i∈[0,1]と特徴ベクトルv_i∈R^8（単位ベクトル）を持つ。
- **構造出力** y = Σ_i w_i·v_i ∈ R^8。**G = ||y − P(t)||²**。
- 乱数：numpy RandomState(seed)。シードs∈{0..29}。全乱数（初期化・P(t)変化・再初期化）は同一RandomStateから既定順で消費。
- **normalize(x) ≡ x/||x||。||x||<1e-9の場合はゼロベクトルを返す**（各所の規定に従いΞ_i=0等として扱う）。**「球面一様単位ベクトル」≡ RandomState.randn(8)をnormalizeしたもの**（第2R・ChatGPT推奨対応）。
- **状態評価タイミング（凍結・第2R・ChatGPT重大対応・提供文言を採用）**：各ステップでは開始時状態から y・G・Ξ_i・Ξ_global・RQ・λ(t)・φ(N)・ψ・∇G を一度だけ算出し、それを全要素で共有してLERIを計算する。更新後の状態は次ステップで初めて評価に使用する。

### 2.2 初期化とコアモチーフ（第1R・ChatGPT重大7対応）
- モチーフ方向 m0 = 球面一様単位ベクトル（ガウス8次元を正規化）。
- **コア要素 i=0..7**：v_i = normalize(m0 + 0.05·g_i)（g_i=標準ガウス8次元）、w_i = 0.8。
- **非コア要素 i=8..31**：v_i = 球面一様単位ベクトル、w_i = 0.5。
- **初期化合格基準（UT0）**：t=0でコア8要素の平均Ξ_i ≥ 0.7。満たさなければg_iを引き直す（最大100回・実測値を記録）。

### 2.3 非定常目標P(t)（第1R・ChatGPT重大8＋Gemini推奨＝共振対策）
- P(0) = 4.0 × 球面一様単位ベクトル。
- 変化イベント間隔Δ：**整数一様乱数[160, 240]**（200固定をやめ揺らぎ化。交互切替周期との共振を物理的に破壊）。
- 変化内容：8次元のうち一様乱数で選んだ4次元に、それぞれN(0,1)×1.0を加算。
- 総ステップT=3000。変化イベント系列はシードごとに事前生成し全条件で共有（対応あり設計）。

### 2.4 LERI式の各項（全て具体式・第1R・ChatGPT重大1〜4対応）
LERI_i = φ(N)·ψ(e_i) − λ(t)·∇G_i + μ·Ξ_i

- **φ(N)（語りの駆動力・縮約版）**：
  - 改善率 imp_t = clip((G_{t−50} − G_t)/(G_{t−50}+1e-9), −1, 1)（t<50はG_0を基準）。
  - Pain：1ステップでGが5%超悪化したイベントeについて pain_t = Σ_e exp(−(t−t_e)/100)、PainLevel = tanh(pain_t/3)。
  - **φ(N) = clip(imp_t − 0.5·PainLevel, −1, 1)**。理論の3成分のうちLNVを実装しない縮約版であることを明示。
- **ψ(e_i)（要素の寄与方向）**：r_i = v_i·(P(t)−y)、**ψ(e_i) = r_i / max_j|r_j|**（全r_j=0なら全ψ=0）。
- **∇G_i**：w_iに関する中心差分。ε=1e-4。g_i = (G(w_i+ε) − G(w_i−ε))/(2ε)、**∇G_i = g_i / max_j|g_j|**（全ゼロなら0）。
- **Ξ_i（自己相似性の操作的代理）**：近傍 = リング位相で i±1, i±2（mod N・計4要素）。局所パターン l_i = normalize(v_i + Σ_近傍 v_j)。縮約パターン m = normalize(Σ_j w_j·v_j)（ノルム<1e-9ならΞ_i=0）。**Ξ_i = max(0, cos(l_i, m))**。
- **λ(t)・RQ・Ξ_global（本書を凍結定義とする・外部擬似コード参照を廃止＝第1R・ChatGPT推奨3対応）**：
  - Ξ_global = mean_i Ξ_i。RQ = clip((G_{t−200} − G_t)/(G_{t−200}+1e-9), 0, 1)（t<200はG_0基準）。
  - **λ(t) = max(0, 0.7 − 0.3·RQ·Ξ_global)**。

### 2.5 更新規則（第1R・ChatGPT重大5対応）
- 判定：LERI_i > +0.1 → 収束実行／LERI_i < −0.1 → 散逸実行／それ以外 → 保持。
- **収束実行**：w_i ← min(1, w_i + 0.05)。v_i ← normalize((1−0.1)·v_i + 0.1·d)、ここでd = normalize(P(t) − y)（||P−y||<1e-9なら回転なし）。
- **散逸実行**：w_i ← 0.5·w_i。v_i ← 新しい球面一様単位ベクトル（同一RandomStateから消費）。
- **同時演算**：全32要素のLERIを同一の状態(y, G, P)で計算してから、全要素の実行を同一ステップ内で適用。
- **交互切替対照**：LERI計算は同時演算と完全同一（毎ステップ・全要素）。ただし**実行**のみ、収束ステップでは収束判定の要素だけ、散逸ステップでは散逸判定の要素だけを適用。周期c∈{1,10,60}（cステップごとに収束/散逸モードを切替。開始は収束モード）。

## 3. 主要仮説（2本のみ・ホルム補正・30シード・17日に1回勝負）

- **P1（μΞ＝選択的な防波堤・差の差検定に変更＝第1R・AIスタジオ重大1対応）**：
  - 生存 ≡ t=3000でw_i > 0.3。コア生存率 S_core = 生存コア数/8。非コア生存率 S_non = 生存非コア数/24。
  - **選択的保護量 D ≡ S_core − S_non** を条件ごとに算出。
  - 検定：**ΔD = D(μ=0.2・同時) − D(μ=0・同時) が正**（対応ありウィルコクソン、pratt、両側、rank-biserial併記、参考t）。
  - 意味：μΞ項が「全体の底上げ」ではなく「自己相似性の高い核を選択的に守る防波堤」であることを、コアと非コアの差の変化で検定する。全体底上げならDは変化しない。
- **P2（同時演算の優位・主張の限定付き＝第1R・Gemini重大対応）**：
  - **累積G改善 ≡ Σ_{t=1..3000} max(0, G_{t−1} − G_t)**（正改善のみの積算・第1R・ChatGPT重大6対応）。
  - 検定：同時演算（μ=0.2）の累積G改善が、**最良の交互対照**（c∈{1,10,60}のうち事後に累積G改善が最大のもの・チェリーピッキング封じとして最も不利な比較を自ら選ぶ）より大きい（対応ありウィルコクソン、pratt、両側）。
  - **主張の限定（凍結）**：有意であっても、その効果には「判定遅延（タイムラグ）の排除効果」が不可分に結合している。機構が純粋な相乗的同時作用か時間遅延の最小化かは本実験では分離せず、**現象として「同時処理が非定常追従において最良対照を凌駕する」事実のみを主張する**。
- 主要検定は2本のみ。ホルム補正。他は一切検定しない。

## 6. 自己申告の穴（更新版・第2R監査者への正直な白状）

2. **【新】P1の残余**：差の差（ΔD）に変更したが、μ=0でLERI分散が縮小し状態遷移頻度自体が下がる効果は残る。ただしこの効果はコア・非コアに同方向に働くため、Dの差分には一次では乗らない、というのが本設計の主張。二次の相互作用（遷移頻度低下がコアと非コアに非対称に効く経路）は分離しない。この扱いで十分か。また、S_core(μ=0)が天井（1.0）に達している場合、ΔDによる効果測定は測定レンジの制約を受ける（第2R・AIスタジオ推奨の採録・仕様変更なし）。
3. **【新】P2の残余（第1R・Gemini重大の受容）**：判定遅延の排除効果と相乗的同時作用は分離不能。主張を現象レベルに限定した（§3）。
4. **【新】UT3とP1の関係（第1R・AIスタジオの問いへの応答）**：UT3は静的・単独項の仕様確認（設計により真）。P1は全項が稼働する動的環境で、かつ差の差として検定するため、UT3の合格からP1の支持は導出されない（φの散逸圧とλ∇Gの追従圧の嵐の中で選択的保護が発現するかは測定によってのみ決まる）。この切り分けで十分か。
